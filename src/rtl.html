RTL

Usage

We are using VueUse Composition Utilities to control RTL (Right-to-Left) in this template. To change the direction, simply click the RTL button in the header, which will call the RTL function and update the template direction based on the user's preference.

header.vue

<script setup lang="ts">

 import { useTextDirection } from "@vueuse/core";
              
  const dir = useTextDirection();
            
  const changeDirection = () => {
  dir.value = dir.value === "rtl" ? "ltr" : "rtl";
};

// Header RTL Button for control RTL function
<button @click="changeDirection()">
   <Icon
     v-if="dir === 'rtl'"
     :size="22"
     :strokeWidth="1.5"
     name="AlignLeft"
     class="text-muted" />
   <Icon v-else :size="22" :strokeWidth="1.5" name="AlignRight" class="text-muted" />
 </button>

The root HTML file must use dir="ltr" in the HTML tag, as per Tailwind CSS instructions

index.html

<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    ......
    ......
  </head>
  <body class="bg-background">
    .....
    .....
  </body>
</html>
How to remove RTL?
  • Remove the dir="ltr" attribute from the HTML tag in the index.html file.
  • Remove the RTL function and RTL button from src/layout/header/Header.vue.
